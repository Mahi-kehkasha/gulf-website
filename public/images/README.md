# Images Folder

Place all your image assets in this folder.

## Recommended Structure:
- `/images/products/` - Product images
- `/images/industries/` - Industry images  
- `/images/gallery/` - Gallery images
- `/images/certificates/` - Certificate images
- `/images/about/` - About page images
- `/images/hero/` - Hero section images (fallback)

## Usage:
Reference images from this folder using: `/images/your-image.jpg`
