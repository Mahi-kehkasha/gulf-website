# About Page Images

## Founder Image

Place the founder's image in this directory with the filename:
- `founder.jpg` - Main founder image (recommended size: 400x400px or larger, square format)

The image will be displayed on the About Us page in the Founder's Message section.

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 400x400px minimum (square format preferred)
- File size: Optimize images to keep file sizes reasonable (< 500KB recommended)

## Usage

The founder image is automatically loaded on the About Us page at `/about`. Make sure the image file is named exactly `founder.jpg` for it to display correctly.
