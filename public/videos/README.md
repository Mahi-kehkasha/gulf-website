# Videos Folder

Place all your video assets in this folder.

## Current Videos:
- `hero-background.mp4` - Main hero background video

## Usage:
Reference videos from this folder using: `/videos/your-video.mp4`

## Video Requirements:
- Format: MP4 (recommended for web)
- Codec: H.264
- Keep file sizes optimized for web performance
- Consider using WebM format as fallback for better compression
